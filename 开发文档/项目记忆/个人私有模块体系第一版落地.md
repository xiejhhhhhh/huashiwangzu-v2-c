框架层面落地个人私有模块体系，新增 3 文件 + 改 7 文件。

关键设计：
- 私有模块走 `data/workspaces/{user_id}/private_modules/{key}/`
- 复用框架统一 loader (`registry.import_module_router`)，不另起炉灶
- 生命周期 API：preview → install → activate → deactivate → rollback → uninstall
- `register_capability` 加 `owner_id` 参数实现 owner 隔离
- LKG（last-known-good）自动保存，失败回滚
- checksum 校验（SHA256 manifest）
- `framework_private_modules` 表自动建表

3 新文件: models/private_module.py, services/private_module_service.py, routers/private_modules.py
7 改文件: models/__init__.py, registry.py, module_registry.py, app_service.py, main.py, migrations/env.py, scan-modules.js
