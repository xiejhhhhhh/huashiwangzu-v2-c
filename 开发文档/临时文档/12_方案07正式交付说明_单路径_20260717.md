## 0. 边界（华哥定）

1. **移除 WP7 / 一切操库迁移计划**（回填、约束 finalize、影子读切换等由其他 agent 负责）。
2. **不做两套代码**：桌面、打开、内容读写只保留一套正式主路径。
3. 当前分支即正式实现，不另起影子系统。

## 1. 正式主路径

### 桌面
- 唯一清单：`GET /api/desktop/products`
- 前端 `loadAppRegistry` **只**拉 Product Catalog
- **不再**合并 `/api/desktop/apps` 到桌面图标

### 打开文件
- 唯一解析：`POST /api/content/open`（Content Open Resolver）
- 前端 `openFileByRecord` **只**走 Resolver
- **不再**降级旧后缀 `file-association-registry` 决定打开器

### 内容
- 读：`content_runtime_service`（canonical 优先，无则现场归一 content_json）
- 写：draft / save / lease / export / publish（`/api/content/*`）
- 摄取：`content_ingest_stage` 异步 DAG（上传只唤醒）

### 产品
首批 10 个 Product（files/office/text/media/ai/knowledge/messages/content-studio/settings/recycle）

## 2. 明确不在本交付

- 数据库回填 / 约束切换 / 迁移批跑
- Knowledge/Viewer 旧旁路的强制下线（属其他 agent / 后续）
- 完整 Office 高保真编辑器（当前是产品壳 + Runtime 会话）
