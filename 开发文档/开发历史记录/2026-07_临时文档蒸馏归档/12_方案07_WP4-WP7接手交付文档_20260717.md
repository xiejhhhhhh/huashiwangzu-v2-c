### 契约（§19.6 OfficeDocumentAdapterV1，冻结）
- Adapter 生命周期：descriptor/createDraft/open/hydrate/edit/getDirtyState/save(expectedVersionId,idempotencyKey,lockToken)/saveAs/export/publish/canClose/renewLock/dispose。
- **Adapter 只能调 Content Runtime，不能接物理路径/不能自己调 Parser**（复用 WP3 `content_runtime_service`）。
- `createDraft` 原子建 source-less Package + 空白 Version，返回 file_id=null。
- 自动保存只新增 Canonical Version，不产生桌面 File。
- 首次显式 Save 才建 File + Artifact + primary ArtifactVersion/FileRevision（Package 仍无 source_file_id）。
- 导入文件保存：SourceFileRevision 永久保留，保存建新 Canonical Version + primary projection 物化为 `origin=projection` 新 Revision，不覆盖导入 Revision。
- 保存重校验权限+lock token+expected_version_id，冲突返回当前版本不自动覆盖。
- **编辑租约**：`framework_content_edit_leases` 已建（每Package最多一个有效租约），配 `content_edit_lease_service.py`。
- 错误码（§19.6 固定表）：PRODUCT_*/CONTENT_*/INGEST_*/DESKTOP_STATE_CONFLICT/IDEMPOTENCY_CONFLICT，HTTP：处理中/只读降级200、异步受理202、版本/幂等冲突409、租约423、权限403、不支持内容422、运行时不可用503。

### ⚠️ 保真门禁（§19.2）
- **导入文件编辑边界按 fidelity 逐文件判定，不按后缀整体放开**。WP3 归一现在导入文件一律 `editable=false`（保守）。WP5 若要放开编辑，必须有往返保真测试证明（Office表格/样式/合并/分页/幻灯片坐标/资源引用往返保真），未达阈值只读。
- Excel：可保留计算引擎，但状态必须能从 CanonicalIR 初始化并提交回统一版本，不能当第二事实源。Excel 迁回是大工程，WP5 若不做则明确标 deferred。

### Gate 5（验收）
草稿不产生File；首次Save产生primary projection；权限变化/租约过期/版本冲突/关闭未保存/刷新恢复/大文件分片加载通过。
